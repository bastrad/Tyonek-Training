## Trade Classification
- **Primary occupation:** Network Security Administrator / Analyst
- **O*NET-SOC code:** 15-1244.00 — Network and Computer Systems Administrators
- **Related occupation(s):** 15-1212.00 Information Security Analysts; 15-1241 Network Architects
- **DOL Registered Apprenticeship:** Apprenticeable (Network/IT occupations)
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=15-1244.00
- **Apprenticeship suitability:** High

## Progression Path
1. **Apprentice Network Technician** — Security+ baseline, device config, monitoring under supervision
2. **Journey Network Security Admin** — DoD 8570 IAT II/III, firewall/IDS ops, hardening (independent)
3. **Senior / Lead Security Engineer** — architecture, zero-trust, mentors technicians

## Typical Apprenticeship Structure
- Term: ~2,000–4,000 OJL hours + Related Technical Instruction
- Completion credential: journey-level admin + DoD 8570/8140 baseline

## Company Sponsorship Notes
- Vendor certs (CCNP Security, PCNSE, NSE) in `CERTIFICATIONS.md` make natural journey milestones.
