## Trade Classification
- **Primary occupation:** Information Security Analyst / Cybersecurity Analyst
- **O*NET-SOC code:** 15-1212.00 — Information Security Analysts
- **Related occupation(s):** 15-1299.05 Information Security Engineers; 15-1244 Network/Systems Admins
- **DOL Registered Apprenticeship:** Apprenticeable (Cybersecurity Support Technician / Information Security Analyst)
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=15-1212.00
- **Apprenticeship suitability:** High (Emerging-but-established)

## Progression Path
1. **Apprentice / Cyber Support Technician** — Security+ baseline, SOC monitoring, tooling under supervision
2. **Journey Cyber Analyst / Operator** — DoD 8570 IAT III (CISSP/CASP+), GPEN/CEH, independent operations
3. **Senior / Master Operator / Lead** — advanced exploitation (OSCP/GXPN), mission lead, mentors apprentices

## Typical Apprenticeship Structure
- Term: ~2,000–4,000 OJL hours + Related Technical Instruction (competency-based common in cyber)
- Completion credential: journey-level analyst with DoD 8570/8140 baseline certs

- Excellent registered-apprenticeship candidate; the DoD 8570/8140 ladder in `CERTIFICATIONS.md` provides ready-made journey gates.
- Clearance (TS/SCI) is the gating prerequisite for most task orders.
