# Cyber Operations — Certifications & Qualifications

**Subsidiary:** Tyonek Mission Support (Tyonek Services Group, Inc.)
**Work Process:** Cyber Operations
**Last updated:**

> **Legend** — This is a U.S. Government contract environment. All regulatory, statutory, and contractually-flowed requirements are listed as **Required**.
> - **Required** — mandated by law, regulation, contract, or needed to legally/safely perform the work.
> - **Recommended** — industry standard; strengthens quality, compliance, and competitiveness.
> - **Nice-to-have** — broadens capability or supports career development.

## Required (DoD 8570.01-M / DoD 8140 workforce baseline)
- Security clearance (typically TS/SCI)
- **IAT Level III** baseline — **CISSP**, **CASP+ CE**, or **CCNP Security**
- **CompTIA Security+ CE** (minimum IAT Level II baseline)
- **GPEN** and/or **CEH** (offensive / penetration-test roles)
- Computing Environment (CE) certification for the operating system/tool in use
- Annual DoD Cyber Awareness / OPSEC training

## Recommended
- Offensive Security **OSCP**
- GIAC **GCIH / GCIA / GCFA** (incident handling / analysis / forensics)
- **CISM** — Certified Information Security Manager

## Nice-to-have
- GIAC **GXPN / GREM** (advanced exploitation / reverse engineering)
- Cloud security (AWS Security Specialty, Azure Security Engineer)
- **CISSP** concentrations (ISSEP/ISSAP)
