> Maps this job scope to the recognized U.S. DOL classification and its progression. Software

## Trade Classification
- **Primary occupation:** Software Developer / IT Implementation Specialist
- **O*NET-SOC code:** 15-1252.00 — Software Developers
- **Related occupation(s):** 15-1232.00 Computer User Support Specialists; 15-1244 Systems Admins
- **DOL Registered Apprenticeship:** Apprenticeable (Software Developer; IT Support)
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=15-1252.00
- **Apprenticeship suitability:** High

## Progression Path
1. **Apprentice Developer / IT Specialist** — coding/config fundamentals, deployment support, ticket handling
2. **Journey Developer / Implementation Specialist** — independent implementation, integration, workforce training
3. **Senior / Lead Engineer / Architect** — solution design, DevSecOps, mentors apprentices

## Typical Apprenticeship Structure
- Term: ~2,000–4,000 OJL hours + Related Technical Instruction (often competency-based)
- Completion credential: journey-level developer/specialist + role-based (Microsoft) certs

## Company Sponsorship Notes
- One of the most common registered-apprenticeship models nationally; ITIL/Microsoft certs from `CERTIFICATIONS.md` support RTI.
