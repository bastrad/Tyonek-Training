- Security clearance (as required)
- **CompTIA Security+ CE** (IAT Level II — required for privileged/IA-enabled roles under DoD 8570/8140)
- **Microsoft** role-based certification(s) relevant to the implemented stack (Azure/M365 administration)
- Section 508 accessibility compliance (federal software) — 29 U.S.C. 794d
- Annual DoD Cyber Awareness training

## Recommended
- **ITIL 4** Foundation (service management)
- **CompTIA A+ / Network+**
- **PMP** (implementation project leads)
- Certified ScrumMaster (CSM) / Agile

## Nice-to-have
- AWS / Azure architect certifications
- DevOps / DevSecOps certification
- Database / platform-specific admin certs
