# Quality Control — Trade Classification & Progression

**Subsidiary:** Tyonek Contractor Services, LLC
**Work Process:** Quality Control
**Last updated:**

> Maps this job scope to the recognized U.S. DOL classification and its progression. Construction
> inspection is apprenticeable; specialty testing is credential-based.

## Trade Classification
- **Primary occupation:** Construction & Building Inspector / QC Manager
- **O*NET-SOC code:** 47-4011.00 — Construction and Building Inspectors
- **Related occupation(s):** 51-9061 Inspectors/Testers; 19-4099 QC testing techs
- **DOL Registered Apprenticeship:** Apprenticeable (Construction Inspector)
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=47-4011.00
- **Apprenticeship suitability:** Moderate–High

## Progression Path
1. **Apprentice / QC Technician** — three-phase inspection support, field testing (ACI concrete), documentation
2. **QC Inspector / Manager (CQM-C)** — runs the contractor QC system, submittals, definable features of work
3. **Senior QC Manager / Lead (CWI, ICC, NICET)** — multi-trade inspection authority, mentors inspectors

## Typical Structure
- Construction Inspector apprenticeship (~4,000–8,000 OJL hrs + RTI)
- Completion credential: journey-level inspector + CQM-C for federal QC-manager roles

## Company Sponsorship Notes
- CQM-C is mandatory for the QC Manager on federal projects; ACI/CWI/ICC certs in `CERTIFICATIONS.md` build the master tier.
