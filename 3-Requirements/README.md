# Phase 3 · Requirements

*What training the law and our contracts **require** — the standard we measure everyone against —
and the process for confirming it's real, not assumed.*

## What's in here
| File / folder | What it is |
| --- | --- |
| [Governing-Standards-Reference.md](Governing-Standards-Reference.md) | **The single catalog** of everything that requires training — every governing standard with plain-language "what it requires," who it applies to, recurrence, and where to obtain it. |
| [Training-and-Certification-Links.md](Training-and-Certification-Links.md) | The official-website directory for every regulation, standard, and certification body. |
| [Requirements-Verification-Process.md](Requirements-Verification-Process.md) | How to confirm each requirement against authoritative sources (contracts, regulation, standards). |
| [templates/Requirements-Verification-Register.md](templates/Requirements-Verification-Register.md) | Fillable form — one row per requirement, with source, citation, and applicability. |
| [public-domain-copies/](public-domain-copies) | Plain-text extracts of public-domain U.S. government regulations. |

## Copyright & sourcing boundary
- **Public-domain material** (U.S. regulations, OSHA/FAA/DoD publications) may be stored here.
- **Copyrighted standards** (AS9100, AS9110, ISO 9001, AWS D17.1, IPC, NADCAP) are **not**
  reproduced — the catalog summarizes them and links to the publisher to buy a licensed copy.
- **Always verify the current revision** at the official source.

## How this phase feeds the rest
The [governing-standards catalog](Governing-Standards-Reference.md) plus each job's own
`CERTIFICATIONS.md` (Phase 2) together form the **required list** — the "answer key" that
[Phase 4](../4-Discover-and-Analyze/) compares the company's current state against.
