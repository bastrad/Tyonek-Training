# Interview Prep

This folder is public on purpose.

For the Training Manager role, I built the framework in this repository to show how I'd approach
the job. I'm also sharing **how I prepared** — a one-page summary and the questions I expect with
my honest answers — because I'd rather you see the real thinking and effort than a polished pitch.

I'm putting this in the open because I mean it: I want to do this work, and I want to do it well.
