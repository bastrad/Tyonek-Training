# Safeguarding & Storage — Trade Classification & Progression

**Subsidiary:** Tyonek Services Group, Inc.
**Work Process:** Safeguarding & Storage (AA&E, HAZMAT, Controlled Data, Comms)
**Last updated:**

> Maps this job scope to recognized classifications. This is a security/property occupation
> with a credential-based ladder rather than a classic trade apprenticeship.

## Trade Classification
- **Primary occupation:** Security Specialist / Property Custodian
- **O*NET-SOC code:** 33-9032.00 — Security Guards (physical security); 13-1199 Business Operations Specialists (security admin)
- **Related occupation(s):** 43-5111 Weighers/Checkers; property & COMSEC custodian roles
- **DOL Registered Apprenticeship:** Emerging (Security-related apprenticeships exist); primarily credential-based
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=33-9032.00
- **Apprenticeship suitability:** Credential-based

## Progression Path
1. **Security/Storage Assistant** — access control, AA&E/HAZMAT storage basics, cleared-material handling
2. **Security Specialist / COMSEC Custodian** — NISPOM/32 CFR 117 program execution, CUI, inventories
3. **Facility Security Officer (FSO) / ISP / Lead** — program ownership, insider threat, audits

## Credential Ladder (in lieu of trade journey levels)
- Entry: clearance + NISPOM training → Mid: FSO certification (CDSE) → Senior: NCMS ISP, CPP
- See `CERTIFICATIONS.md`.

## Company Sponsorship Notes
- Run as a certification-based security career ladder; a registered apprenticeship is optional and would be competency-based.
