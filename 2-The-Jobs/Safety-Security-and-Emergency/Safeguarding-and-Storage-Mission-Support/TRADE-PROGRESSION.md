# Safeguarding & Storage — Trade Classification & Progression

**Subsidiary:** Tyonek Mission Support (Tyonek Services Group, Inc.)
**Work Process:** Safeguarding & Storage (Data, Materials, Equipment, Clearances)
**Last updated:**

> Maps this job scope to recognized classifications. Security occupation with a credential-based
> ladder rather than a classic trade apprenticeship.

## Trade Classification
- **Primary occupation:** Industrial Security Specialist
- **O*NET-SOC code:** 13-1199.00 — Business Operations Specialists, All Other (security admin); 33-9032 Security Guards (physical)
- **Related occupation(s):** COMSEC custodian; Insider Threat program roles
- **DOL Registered Apprenticeship:** Emerging (Security-related); primarily credential-based
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=33-9032.00
- **Apprenticeship suitability:** Credential-based

## Progression Path
1. **Security Assistant** — access control, CUI handling, inventories, cleared-material basics
2. **Security Specialist / COMSEC Custodian** — NISPOM/32 CFR 117 program execution, personnel security
3. **Facility Security Officer (FSO) / ISP / Lead** — program ownership, insider threat, audits

## Credential Ladder
- Entry: clearance + NISPOM training → Mid: FSO certification (CDSE) → Senior: NCMS ISP, CPP
- See `CERTIFICATIONS.md`.

## Company Sponsorship Notes
- Run as a certification-based security career ladder; a registered apprenticeship would be competency-based.
