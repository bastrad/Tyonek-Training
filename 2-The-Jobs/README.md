# Phase 2 · The Jobs

*The work we train. All 46 jobs, grouped by **trade/discipline** (not by company). Each job has
its own folder with three files:*

- **`README.md`** — the training document (what the job is, how it's done safely, sign-off)
- **`CERTIFICATIONS.md`** — the certifications it needs (Required / Recommended / Nice-to-have)
- **`TRADE-PROGRESSION.md`** — the career path (trade + apprentice → journeyman → master)

Every job's page notes which **company** it belongs to. For a one-page rollup of all 46 jobs
and their trades, see the [Trade & Progression Index](Trade-and-Progression-Index.md).
