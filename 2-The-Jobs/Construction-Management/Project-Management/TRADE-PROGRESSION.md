## Trade Classification
- **Primary occupation:** Construction Manager / Project Manager
- **O*NET-SOC code:** 11-9021.00 — Construction Managers
- **Related occupation(s):** 13-1082 Project Management Specialists; craft-trade journeyworkers advancing to super
- **DOL Registered Apprenticeship:** Emerging (Construction Manager); often reached via craft apprenticeship + experience
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=11-9021.00
- **Apprenticeship suitability:** Credential-based (with craft-trade feeder path)

## Progression Path
1. **Field/Project Engineer** — submittals, RFIs, scheduling support, QC/safety coordination
2. **Project Manager / Superintendent (CQM-C, OSHA 30)** — runs projects, three-phase QC, cost/schedule
3. **Senior PM / Program Manager (PMP/CCM)** — portfolio delivery, client interface, mentors PMs

## Credential Ladder
- Entry: OSHA 30 + EM 385-1-1 → Mid: CQM-C, Alaska GC familiarity → Senior: PMP, CMAA CCM
- See `CERTIFICATIONS.md`.

## Company Sponsorship Notes
- A journey-level craft worker (e.g., carpenter, electrician) is a natural feeder into this ladder.
- CQM-C is required to run QC on federal (USACE/NAVFAC) projects.
