# Network Security — Certifications & Qualifications

**Work Process:** Network Security
**Last updated:**

> **Legend** — This is a U.S. Government contract environment. All regulatory, statutory, and contractually-flowed requirements are listed as **Required**.
> - **Required** — mandated by law, regulation, contract, or needed to legally/safely perform the work.
> - **Recommended** — industry standard; strengthens quality, compliance, and competitiveness.
> - **Nice-to-have** — broadens capability or supports career development.

## Required (DoD 8570.01-M / DoD 8140 baseline)
- Security clearance (as required)
- **IAT Level II/III** baseline — **Security+ CE**, **CISSP**, or **CASP+ CE**
- Computing Environment (CE) certification for the security appliances in use
- Annual DoD Cyber Awareness / OPSEC training

## Recommended
- **CCNP Security** / **CCNA Security**
- Palo Alto **PCNSE**
- Fortinet **NSE 4+**
- **CISM**

## Nice-to-have
- **CISSP** concentrations (ISSAP)
- Cloud security specialty (AWS/Azure)
- Zero Trust architecture training
