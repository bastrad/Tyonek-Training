## Trade Classification
- **Primary occupation:** Network / Systems Engineer (Cyber Range)
- **O*NET-SOC code:** 15-1241.00 — Computer Network Architects
- **Related occupation(s):** 15-1231.00 Computer Network Support Specialists; 15-1244 Network/Systems Admins; 15-1212 Info Security Analysts
- **DOL Registered Apprenticeship:** Apprenticeable (IT / Network occupations)
- **Occupation finder:** https://www.apprenticeship.gov/apprenticeship-occupations/listings?occupationCode=15-1241.00
- **Apprenticeship suitability:** High

## Progression Path
1. **Apprentice Range Technician** — virtualization, provisioning, scenario setup under supervision
2. **Journey Range Engineer** — designs/operates range environments, builds scenarios independently
3. **Senior / Lead Range Architect** — range architecture, integration, mentors technicians

## Typical Apprenticeship Structure
- Term: ~2,000–4,000 OJL hours + Related Technical Instruction
- Completion credential: journey-level network/systems engineer + Security+ baseline

## Company Sponsorship Notes
- Pairs with the cyber-operations apprenticeship; RTI via vendor certs (CCNP, VCP, RHCE) from `CERTIFICATIONS.md`.
